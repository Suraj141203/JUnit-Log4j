# Log4j with JUnit 4  Project-Simple

## 📌 Overview
This project demonstrates how to integrate **Log4j** logging framework with **JUnit 4** for unit testing in Java applications.

It shows:
- How to configure Log4j
- Different logging levels (INFO, DEBUG, ERROR, etc.)
- Writing and executing JUnit 4 test cases
- Logging test execution results
- Best practices for structured logging in Java projects

---

## 🛠 Technologies Used

- Java (JDK 8+)
- Log4j (Logging Framework)
- JUnit 4 (Unit Testing Framework)
- Maven / Gradle (Optional – if used)
